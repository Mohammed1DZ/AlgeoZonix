# **App Name**: OrderFlow

## Core Features:

- User Authentication: Secure user authentication using Firebase Auth with email/phone login.
- Role-Based Access Control: Implement RBAC to restrict access based on user roles (Client/Admin).
- Order Submission: Structured order submission form for clients.
- Order Tracking: Real-time order status tracking with clear progress indicators for clients.
- Admin Order Management: Admin dashboard to view, filter, and manage all client orders.
- Notifications: Automated email/SMS notifications triggered by order submissions and status changes.
- Summary Reporting: Ability to generate and export summary reports (CSV/JSON) for admin users.

## Style Guidelines:

- Primary color: Deep Blue (#2E4057) to convey trust and professionalism, matching the AlgeoZonix logo.
- Background color: Light gray (#E0E5EC), offering a clean, modern feel.
- Accent color: Teal (#008080) for interactive elements and highlights, aligning with the AlgeoZonix logo.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined, objective, neutral look, suitable for headlines or body text.
- Simple, consistent icons from Material Design to enhance usability.
- Clean and responsive layout using Material UI components.
- Subtle animations for status updates and transitions.
